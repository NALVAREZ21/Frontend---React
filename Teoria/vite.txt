PASO A PASO PARA CREAR UNA APLICACION DE REACT CON VITE

1)_ VERIFICAR VERSION DE NODE CON node -v
2)_ VERIFICAR VERSION DE NPM CON npm -v
3)_ REALIZAR npm create vite@latest
4)_ LUEGO DEL PASO 3 PONER PROJECT NAME
5)_ SELECCIONAR EL FRANMEWORK (REACT)
6)_ SELECCIONAR UNA VARIANTE (JAVASCRIPT)
7)_ ACCEDER A LA CARPETA CREADA (con el comando cd) y 
luego hacer npm i (ASI INSTALA NODE_MODULES) 
8)_ PARA LEVANTAR LA APP, PONER npm run dev
